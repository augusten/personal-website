Bachelor thesis in algorithmic game theory

The title of the thesis was "The Empirical Inefficiency of Equilibria in Coordination Games on Social Networks", and my aim was to experimentally investigate the aforementioned games with the help of real-world Facebook, scale-free and random model networks. During the experimentation part I:

  • Pre-processed the data from Facebook networks using Python and the Pandas library
  • Processed data and modeled scale-free and random networks using NetworkX
  • Applied best-response and greedy algorithms to create the coordination game set-up

The link directs to the GitHub repository containing the code.
